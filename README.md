# 🧘 Detoxify – Focus, Reflect, and Grow

Detoxify is a wellness-focused productivity app built with **Next.js**. It empowers you to build better habits, reflect through journaling, and stay on track with a minimalist timer.

## 🌟 Features

- ⏱️ **Timer** – Use a calm, responsive timer to stay focused and mindful.
- 📔 **Journal** – Write down your thoughts and feelings to reflect on your day.
- 📋 **Habit Tracker** – Add and monitor your daily habits to build consistency.
- 🎨 **Elegant UI** – A soothing design inspired by soft pastel gradients and clean typography.
